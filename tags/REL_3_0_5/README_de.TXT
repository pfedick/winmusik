Patrick F.-Productions - WinMusik 3
===================================

WinMusik ist ein Programm zum Verwalten von Musiktiteln auf
Audio-CDs, DVDs, Musikkassetten, Schallplatten und anderer
Tonträger.

  - Komfortable Oberfläche zum Eingeben und Editieren
    Ihrer Musiktitel.
  - Import von MP3-Dateien
  - ID3v2-Tag-Support
  - Umfangreiche Suchfunktionen zum Auffinden bestimmter
    Titel oder Interpreten.
  - OpenSource unter der GNU GENERAL PUBLIC LICENSE
  - Läuft unter Windows, Linux, FreeBSD und anderen Unixen
