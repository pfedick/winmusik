WinMusik

WinMusik ist ein Programm zum Verwalten von Musiktiteln auf Audio-CDs, DVDs,
Musikkassetten, Schallplatten, Festplatten und anderer Tonträger. Es hat einen
komfortablen Editor und ein einfaches Suchinterface.

  - Komfortable Oberfläche zum Eingeben und Editieren
    Ihrer Musiktitel.
  - Import von MP3-Dateien auf der Festplatte
  - ID3v2-Tag-Support
  - unterstützt Coverbilder in MP3-Dateien
  - Umfangreiche Suchfunktionen
  - CD-Cover drucken
  - Suchlisten
  - OpenSource unter der GNU GENERAL PUBLIC LICENSE
  - Läuft unter Windows XP/Vista/7, Linux, FreeBSD und anderen Unixen
